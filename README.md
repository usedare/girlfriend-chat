# 百变女友

六种风格 AI 恋爱陪伴，支持热切换人设、流式对话、本地记忆持久化。

## 使用方式

1. 打开 [在线版本](https://usedare.github.io/girlfriend-chat/)
2. 点击 ⚙️ 输入 MIMO API Key
3. 点击 🔄 选择喜欢的人设风格
4. 开始聊天！

## 六种人设

- 🎀 萝莉 — 软糯黏人、奶气撒娇
- ☀️ 甜妹 — 元气阳光、治愈甜心
- 👑 御姐 — 成熟宠溺、温柔掌控
- 🌙 御萝双修 — 日常软萌、危机御姐
- 🔥 可盐可甜 — 日常甜蜜、严肃利落
- 💋 甜欲 — 慵懒暧昧、纯欲氛围

## 技术栈

- 纯静态 HTML（单文件）
- 小米 MIMO OpenAI 兼容接口
- 流式 SSE 输出
- LocalStorage 持久化
- 响应式设计（桌面+移动端）
